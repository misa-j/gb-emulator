# Game Boy emulator (DMG)

Game Boy emulator written in C.

## Building and running the project
### Prerequisites

- GCC
- SDL2

## Compile
```bash
gcc -o emu main.c -lSDL2
```
## Run
```bash
./emu /path/to/your/rom.gb
```
